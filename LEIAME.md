# Implementação de Autômato Finito Deterministico (AFD)

## Requisitos

- JDK 21

## Como rodar

- va para o diretorio `src`

```bash
cd src
```

- compile os arquivos java

```bash
javac *.java
```

- rode o App

```bash
java App.java
```

- Utilize os exemplos de inputs e palavras do [arquivo de exemplos](./examples.md)
